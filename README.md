# Quantium retail analytics project
This project analyzes customer purchasing behavior and evaluates store trial performance using retail transaction data. The analysis focuses on customer segmentation, sales trends, brand preferences, and trial store effectiveness.

## Task 1 – Customer Analytics
### Project Overview(TASK1)

This project evaluates the performance of trial stores by comparing them with selected control stores using retail sales data. The analysis measures total sales, customer count, and transaction behavior to determine whether the store trials were successful. Statistical comparison and similarity measures were used to identify control stores and assess the impact of the trial period on store performance.


## Task 2- store trial Analysis
### Project Overview(TASK2)

This project evaluates the performance of trial stores by comparing them with selected control stores using retail sales data. The analysis measures total sales, customer count, and transaction behavior to determine whether the store trials were successful. Statistical comparison and similarity measures were used to identify control stores and assess the impact of the trial period on store performance.

 ## Objectives of TASK1

- Analyze customer purchasing behavior in the chips category
- Identify customer segments based on lifestage and premium spending habits
- Study the impact of brand preference and pack size on purchasing patterns
- Perform data cleaning and exploratory data analysis (EDA)
- Generate insights to support strategic retail recommendations

 ## Objectives OF TASK2

- Evaluate the performance of trial stores against control stores
- Compare total sales revenue, customer count, and transactions per customer
- Identify suitable control stores using similarity and correlation analysis
- Measure the effectiveness of store trials during the trial period
- Determine whether sales growth is driven by more customers or higher purchase frequency
